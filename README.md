## 🏷️ Sobre

O **Gowww** é uma aplicação feita para acompanhar placares ao vivo. O usuário poderá acompanhar os jogos, visualizar informações dos clubes, jogadores, partidas e até mesmo campeonatos.

## 🌈 Interface
